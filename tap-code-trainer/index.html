Tap code trainer

This is a practice thing I made for me and my friend for decoding tap code. Make sure to have sound turned on, then use the button to play a word. I also made an easier version that plays one letter at a time.


Cheat sheet
1 2 3 4 5
1 A B C D E
2 F G H I J
3 L M N O P
4 Q R S T U
5 V W X Y Z

Every letter is knocked in two groups, first y, then x. This means the first column can be memorized and counted along during the first group, and then the following part of the alphabet can be counted during the second, to get to the letter that's knocked.

C (. ...) also represents K, because there isn't enough space for all 26 letters.
