My favorite browser extensions

I don't know of too many browser extensions, but the ones I do know are good, and you should probably install them. I have all of these on my desktop and phone! Also, you should use the Zen browser on my PC. It's basically Firefox but it looks cooler and has some nice features.

  • Ublock Origin - The best adblocker I think. If you don't have adblock for youtube, is it really worth it to still watch youtube? (no)

  • Dark reader - Not good for some websites, but ouchie owie light mode

  • Unhook - Removes all recommendations on youtube, alternatively only some of them. I've got subscriptions and loads of watch later anyways :)

  • DeArrow - This makes youtube thumbnails and titles user submitted to prevent clickbait. I like it.

  • Sponsorblock - Time is money and watching sponsored segments give creators money indirectly. But ew

  • FontFinder - You can select fonts and right click to see what they are. Good when you see a really sexy typeface

  • YT-shorts block - They're no fun, are they?

Rest in peace Grepper, my formerly second favorite extension :,( The idea was basically that users could submit an answer to a question which would show up before any results if someone googled roughly that question. This was most useful for code, because you could search for reverse list js or something and figure out how to do it without having to click on any link and accept cookies on MDN web docs or whatever. About 2025-5-12 it stopped working, and as far as I know there hasn't been any explanation from the developers.
